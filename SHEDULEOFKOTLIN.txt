## ✅ 2️⃣ RecyclerView (List UI)

### Step-1: Create a layout row:

`item_row.xml`
Contains text + image or anything.

### Step-2: Make a data class

```kotlin
data class Product(val title: String)
```

### Step-3: Adapter

```kotlin
class ProductAdapter(val list: List<Product>) :
    RecyclerView.Adapter<ProductAdapter.ViewHolder>() {

    class ViewHolder(itemView: View) : RecyclerView.ViewHolder(itemView)

    override fun onCreateViewHolder(parent: ViewGroup, viewType: Int): ViewHolder {
        val view = LayoutInflater.from(parent.context)
            .inflate(R.layout.item_row, parent, false)
        return ViewHolder(view)
    }

    override fun onBindViewHolder(holder: ViewHolder, position: Int) {
        holder.itemView.findViewById<TextView>(R.id.title).text = list[position].title
    }

    override fun getItemCount() = list.size
}
```

# ✅ 1️⃣ SharedPreferences

(For saving login, settings, tokens)

```kotlin
val sp = getSharedPreferences("user", MODE_PRIVATE)
sp.edit().putString("username", "Kirito").apply()
```

Retrieve:

```kotlin
val name = sp.getString("username", "")
```

# ✅ 2️⃣ Simple Database (Room)

### Define Entity

```kotlin
@Entity
data class User(
    @PrimaryKey(autoGenerate = true) val id: Int,
    val name: String
)
```

### DAO

```kotlin
@Dao
interface UserDao {
    @Insert
    fun insert(user: User)

    @Query("SELECT * FROM user")
    fun getAll(): List<User>
}
```
